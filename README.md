# DVS_CAD

AI text-to-CAD in the browser. Describe a mechanical part in plain English; Claude generates parametric geometry with true boolean subtraction (BSP CSG engine). Three.js viewport, editable cutters, STL export.

**Live:** https://dvs-cad.vercel.app

Single-file app — `index.html` is the whole thing. Pushes to `main` auto-deploy via Vercel.
